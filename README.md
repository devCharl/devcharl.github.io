# devcharl.github.io

Charles 的个人博客静态站点，由 [GitHub Pages](https://pages.github.com/) 托管。

**在线访问**：[https://devcharl.github.io/](https://devcharl.github.io/)

## 说明

本仓库内容由源码仓库 [blog-source](https://github.com/devCharl/blog-source) 通过 GitHub Actions 自动构建并发布。

请勿直接在此仓库修改 HTML 文件——下次部署会被覆盖。请在 `blog-source` 中编辑 Markdown 后 `git push`。

## 技术栈

- [Hugo](https://gohugo.io/) 0.163+ extended
- [FixIt](https://github.com/hugo-fixit/FixIt) v0.4.5
