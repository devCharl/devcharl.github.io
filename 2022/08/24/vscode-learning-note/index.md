# VS Code 正则表达式与快捷键


## 正则表达式

| 用途 | 表达式 |
|------|--------|
| 删除空行 | `^\s*(?=\r?$)\n` |

## 常用快捷键

| 操作 | 快捷键 |
|------|--------|
| 打开键盘快捷键设置 | `Ctrl+K Ctrl+S`，或 File → Preferences → Keyboard Shortcuts |
| 当前行快速换行 | `Ctrl+Enter` |
| 快速删除当前行 | `Shift+Delete` 或 `Ctrl+Shift+K` |
| 查找 | `Ctrl+F` |
| 文件中查找 | `Ctrl+Shift+F` |
| 替换 | `Ctrl+H` |
| 文件中替换 | `Ctrl+Shift+H` |
